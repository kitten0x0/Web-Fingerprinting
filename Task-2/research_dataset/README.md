#  Research Dataset (Raw PCAPs)

Due to GitHub's strict file size limitations, the raw packet capture (`.pcap`) files generated during Task 2 cannot be hosted in this repository. 

The complete, high-fidelity dataset collected during this research phase totals approximately **40 GB** consisting of 2,400 individual traffic traces.

Instead of uploading the binary files, **screenshots of the directory structure and capture process have been provided** as proof of successful execution and data gathering.
